# MyGarage 🚗

Лично приложение за следене на гаранции и автомобилни документи.

---

## 📱 Как да инсталираш APK на телефона

### Стъпка 1: Качи файловете в GitHub

1. Влез в **github.com** с твоя акаунт
2. Натисни зеления бутон **"New"** (горе вляво)
3. Въведи **Repository name:** `mygarage`
4. Избери **Private** (само ти да го виждаш)
5. Натисни **"Create repository"**

### Стъпка 2: Качи файловете

1. На страницата на новото repository, натисни **"uploading an existing file"**
2. Провлачи (drag & drop) **цялата папка mygarage** в браузъра
3. Напиши съобщение: `Initial commit`
4. Натисни **"Commit changes"**

### Стъпка 3: Изчакай GitHub да направи APK файла

1. Натисни таба **"Actions"** в горното меню
2. Ще видиш работа наречена **"Build APK"** — изчакай да стане зелена ✅ (около 5-10 минути)
3. Натисни на завършената работа
4. Долу виж **"Artifacts"** → натисни **"mygarage-apk"**
5. Свали ZIP файла → разархивирай → вътре е **app-release.apk**

### Стъпка 4: Инсталирай на Android телефона

1. Прати APK файла на телефона (WhatsApp, имейл, или USB кабел)
2. Отвори файла от телефона
3. Ако питa за разрешение: **"Настройки → Инсталиране от неизвестни източници → Разреши"**
4. Натисни **"Инсталирай"**

---

## 🔔 Как работят известията

- При всяко стартиране на приложението, известията се обновяват
- Ще получиш известие **X дни преди изтичане** (X се задава в Настройки)
- Известията работят дори когато приложението е затворено

## 📂 Структура на файловете

```
mygarage/
├── lib/
│   ├── main.dart                    # Главен файл
│   ├── theme.dart                   # Цветове и дизайн
│   ├── models/
│   │   ├── warranty.dart            # Модел за гаранция
│   │   └── vehicle.dart             # Модели за кола и документи
│   ├── services/
│   │   ├── database_service.dart    # Локална база данни
│   │   └── notification_service.dart # Известия
│   └── screens/
│       ├── warranties_screen.dart   # Списък с гаранции
│       ├── warranty_form_screen.dart # Форма за гаранция
│       ├── vehicles_screen.dart     # Списък с коли
│       ├── vehicle_form_screen.dart # Форма за кола
│       ├── vehicle_detail_screen.dart # Детайли на кола + документи
│       ├── document_form_screen.dart # Форма за документ
│       └── settings_screen.dart     # Настройки
├── android/
│   └── app/src/main/
│       └── AndroidManifest.xml      # Android разрешения
├── .github/workflows/
│   └── build.yml                    # Автоматичен билд
└── pubspec.yaml                     # Пакети/зависимости
```
